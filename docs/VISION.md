# Centauri Vision

Centauri is a lo-fi first-person exploration game set on an unknown planet.

The target feeling is closer to Proteus than a survival/crafting game: wandering, noticing, listening, and gradually finding strange systems rather than being pushed through objectives.

## Pillars

- **Exploration first**: walking, looking, and discovering are the core loop.
- **Lo-fi alien beauty**: low-poly geometry, flat shading, pixel-ish UI, limited palettes, fog, silhouettes, and odd colour choices.
- **Trippy but readable**: psychedelic behaviours are welcome, but the player should still understand where they are and how to move.
- **Reactive ambience**: objects should eventually contribute to sound/music based on proximity, time, weather, or player route.
- **Small systems, visible effects**: prefer tiny interactions that alter the world over menus and stats.
- **Agent-friendly changes**: every nontrivial PR should include a deterministic demo path or update the existing PR flythrough.

## Non-goals for now

- Combat.
- Inventory/crafting.
- Dialogue trees.
- Realistic rendering.
- Large asset pipelines.
- Multiplayer.

## Current prototype

The initial scaffold contains a deterministic alien-island flythrough behind `?demo=pr`, plus manual WASD/drag exploration.
